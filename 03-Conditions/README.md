# Conditions

This folder contains examples and exercises demonstrating conditional statements in C#.

## Topics Covered

### if Statement
- Execute code when a condition is true.

### if...else Statement
- Choose between two different code blocks.

### if...else if...else Statement
- Evaluate multiple conditions.

### Nested if
- Place an if statement inside another if statement.

### switch Statement
- Execute different code blocks based on a value.

## Purpose

Understand how conditional statements control the flow of a program by making decisions based on different conditions.
