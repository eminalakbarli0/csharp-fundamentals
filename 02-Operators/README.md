This folder contains examples and exercises demonstrating operators in C#.

## Topics Covered

### Arithmetic Operators
- +
- -
- *
- /
- %
- ++
- --

### Assignment Operators
- =
- +=
- -=
- *=
- /=
- %=

### Comparison Operators
- ==
- !=
- >
- <
- >=
- <=

### Logical Operators
- &&
- ||
- !

## Purpose

Understand how operators are used to perform calculations, compare values, and control program logic.
